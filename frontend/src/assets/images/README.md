# 图片资源目录

## 需要准备的图片资源

### Logo 和品牌图标
- `logo.png` - 网站 Logo
- `instagram.svg` - Instagram 图标
- `facebook.svg` - Facebook 图标

### 首页图片
- 双人模特图片（Hero Section）
- 产品系列封面图（Collections）
- 责任商业部分图片
- 本月畅销产品图片

### 导航图标
- 左右箭头图标
- 放大/缩小图标
- 关闭按钮图标

## 图片规范

### 尺寸建议
- Logo: 200x60px
- Hero 图片: 600x800px
- 产品封面: 400x500px
- 文章图片: 800x1000px

### 格式要求
- 支持格式: JPG, PNG, WebP
- 建议压缩以优化加载速度
- 透明背景使用 PNG 格式

### 命名规范
- 使用小写字母和连字符
- 例如: `product-ring-01.jpg`

## 占位图片

开发阶段使用 `https://via.placeholder.com/` 生成占位图片。

上线前请替换所有占位图片为实际产品图片。
